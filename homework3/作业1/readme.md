&#x20;    整体的py文件批注：

1.dataset.py          定义锥桶数据的类，主要是存储了他每个图片的路径，以及每个照片颜色所对应的标签数字涉及路径的拼装,从而确保每个图片都有对应的标签，这个类下还有获取图片数量（方便打印）函数与将图片装换成RGB格式的函数，最后返回图片及其标签。其次是对图片的基本处理（ai给的），首先是统一分辨率，其次翻转，随即旋转10度，最后是转换成张量还有归一化（跟多元线性回归相似），说是大图片集下三种颜色的经验值，测试集则没有翻转，旋转操作。最后是获取文件路径和设置一次读取图片数量和数据是否打乱（训练集和测试集）

2.model.py

相当于CNN核心函数

&#x20;     首先RGB的通道数为3，卷积层1选择的特征数是16（本来不太理解，后来大概明白就是找类似16个不同的矩阵去判断每个部位的特征关系），相当于输出16个特征（通道），卷积核为3\*3（常用），padding=1，此时补一圈0，最后输出还是原图的尺寸

&#x20;     其次归一化，整理输出的数值，使其稳定分布，防止后面学习尺度太大，通道数与上一个输出通道保持一致

&#x20;     激活函数，使负数变零，正数不变，能够使用非线性的连接方式，不只是线性连接

&#x20;     池化层，为了防止有的特征多余，采用2\*2窗口，相当于保留2\*2矩阵中最大的数，即最强的特征，较少不必要的计算，宽高减一半

&#x20;    展平层，将3D数据转换成1D数据，保证全连接层（读取1D)

&#x20;      全连接层，接受一条线通道\*宽\*高  最后算出每一部分的加权占比对结果判断的影响，分两次特征->要点->分类

3.train.py      训练模型

制定了我从windows复制过来的数据集dataset,还有每次取32张图训练，共20轮，步长0.001（训练后更新参数），打包数据放置模型后创建网络。（ai）调用损失函数和优化器进行学习，下面则是循环训练（20次），指定训练模式，定义正确数量和训练总数，训练->预测结果（前向传播）->计算损失率->梯度清零->反向传播（更新参数）->统计数据，20次完毕则保存模型

4.test.py      测试模型

获取dataset.py中的两个测试模型，创建网络读取训练好的模型，接着开始工作，换为测试模式，关闭梯度和随机丢失神经元，进行调用打印。
